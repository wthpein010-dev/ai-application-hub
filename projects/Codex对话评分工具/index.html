C Codex 对话评分工具Conversation Review & Validator

检查 Codex 对话质量，并导出评分 Excel

读取本机 Codex 会话数据，按提示词清晰度、上下文完整度、可执行性和风险控制进行评分，生成逐条反馈、会话汇总和评分规则工作簿。工具默认会隐藏明显的 API key、token、password 类敏感字符串。

Windows 双击运行 macOS 运行脚本 查看 README 示例评分结果
核心能力

评分输出

运行后会在 `outputs` 文件夹生成 `codex_conversation_review_*.xlsx`，包含总览、逐条评分、会话汇总和评分规则四个工作表。

逐条评分

针对每条用户提示给出分数、反馈、建议改写和原始文本。

会话汇总

按线程聚合提示数量、平均分、主题和可改进方向。

Windows

免 Python 运行

已包含 PyInstaller 打包版本，可直接双击批处理运行。

Windows发布包/Codex对话评分工具/run_codex_review.bat
macOS / Linux

Python 源码运行

使用系统 Python 运行源码版，适合非 Windows 电脑。

python3 ./codex_conversation_reviewer.py
校验工具

会话健康检查

`codex_conversation_validator.py` 可检查 Codex 会话文件是否可读、非空、JSONL 有效，并定位指定 thread id。

python3 ./codex_conversation_validator.py --current
注意

隐私与兼容

  • 工具只读取本地 Codex 会话数据，不会主动上传内容。
  • Windows 可直接运行发布包；macOS 使用 Python 源码方式运行。
  • 如果 Codex 数据目录不在默认位置，可用 `--codex-home` 或 `--scan-root` 指定。
